# 📦 쿠핑(Cuping) 새벽배송 물류센터 RAG 챗봇

사내 규정, 배송 정책, 업무 절차 등의 문서를 기반으로  
정확하고 빠른 질의응답을 제공하는 RAG(ChatGPT + Vector Store) 기반 사내 챗봇입니다.

물류센터 직원들이 업무 중 궁금한 사항을 즉시 조회할 수 있도록 제작되었습니다.

---

## 🚀 주요 기능

- PDF 문서 자동 분석 및 분할  
- OpenAI Embeddings + FAISS 벡터스토어 구축  
- RAG 기반 문서 기반 답변 생성  
- Streamlit 웹 UI 제공  
- 문서 내용이 없으면 정확히 알려주는 안전한 구조  
- OpenAI API 키는 `.env` 로 안전하게 관리
